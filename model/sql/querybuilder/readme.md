# querybuilder SQL语句拼接器

提供拼接sql语句的接口，支持设置sql类型生成不同的语句

## 支持的语法 

### SELECT

### INSERT

### UPDATE

### DELETE

### MISC

## 支持的数据库及类型
### mysql
驱动:github.com/go-sql-driver/mysql

### sqlite
驱动:github.com/mattn/go-sqlite3

### posgresql
驱动:github.com/lib/pq

### mssql
驱动:github.com/denisenkom/go-mssqldb
